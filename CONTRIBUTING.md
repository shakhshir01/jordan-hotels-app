Getting started:
- Fork the repo and create a feature branch off `main`.
- Run `cd jordan-hotels-app && npm install` and verify `npm run dev` works.
- Write tests for any new feature and run `npm test`.

Standards:
- Run `npm run lint` and `npm run format` before committing.
- Commit small, focused PRs with clear descriptions and screenshots when applicable.

CI:
- PRs will run lint, tests and build via GitHub Actions; fix issues reported by the CI before merging.

Creating PRs:
- Use the template in `.github/PULL_REQUEST_TEMPLATE.md` and link related issues.
